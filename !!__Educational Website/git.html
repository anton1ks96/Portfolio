ЧТО ТАКОЕ GIT:

Git — это современная распределённая система контроля версий, которая позволяет отслеживать изменения в файлах, управлять версиями проектов и эффективно работать в команде. Она используется разработчиками для сохранения истории изменений кода, отката к предыдущим версиям и упрощения совместной работы. Git поддерживает как локальное, так и удалённое хранение репозиториев, что делает его незаменимым инструментом для управления проектами любого масштаба.

УСТАНОВКА GIT НА WINDOWS

Скачивание установочного файла

  1. Откройте браузер и перейдите на официальный сайт Git: https://git-scm.com.
  2. На главной странице автоматически будет предложена версия для вашей операционной системы. Нажмите кнопку "Download for Windows".
  3. Дождитесь завершения загрузки установочного файла.

Настройка установщика

При запуске установщика выполните следующие шаги:

  1. Лицензионное соглашение: Нажмите Next, чтобы принять лицензию.
  2. Выбор папки установки: Оставьте путь по умолчанию или выберите другую папку, затем нажмите Next.
  3. Компоненты установки: Оставьте настройки по умолчанию и нажмите Next.
  4. Меню "Пуск": Оставьте название по умолчанию или снимите галочку, если не хотите добавлять ярлык в меню "Пуск", затем нажмите Next.
  5. Выбор редактора по умолчанию: Рекомендуется выбрать vim, но можете выбрать любой предпочитаемый редактор (например, Notepad++), затем нажмите Next.
  6. Настройка имени главной ветки в новых репозиториях: Выберите пункт Override the default branch name for new repositories и укажите main в качестве имени главной ветки. Это имя часто используется в современных проектах вместо master для большей универсальности и соответствия текущим стандартам разработки.
  7. Настройка начального пути (PATH): Выберите опцию Git from the command line and also from 3rd-party software (рекомендуется для универсальности), затем нажмите Next.
  8. SSH-экспорт: Оставьте настройку по умолчанию и нажмите Next.
  9. Настройки HTTPS: Выберите Use the OpenSSL library и нажмите Next.
  10. Выбор конечных настроек: Оставьте все параметры по умолчанию, нажимая Next, пока не увидите кнопку Install.
  11. Нажмите Install для начала установки.

Завершение установки

  1. После завершения процесса установки убедитесь, что отмечена галочка Launch Git Bash, и нажмите Finish, чтобы запустить терминал Git Bash.

УСТАНОВКА GIT НА MACOS

Установка Homebrew

Homebrew — это менеджер пакетов для macOS, который позволяет легко устанавливать, обновлять и управлять программным обеспечением и утилитами через терминал. Он упрощает работу с инструментами разработки, такими как Git, Node.js и Python.

  1. Откройте приложение Terminal на вашем Mac.
  2. Введите следующую команду, чтобы начать установку Homebrew:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)
  3. Дождитесь завершения процесса установки. Во время установки вас могут попросить ввести пароль администратора.
  4. После завершения проверьте, что Homebrew установлен, выполнив:
    brew --version
  5. Если установка прошла успешно, вы увидите установленную версию Homebrew, например:
    Homebrew 4.x.x

Установка Git через Homebrew

  1. В терминале выполните команду:
    brew install git
  2. Дождитесь завершения установки

Проверка установки

  1. Убедитесь, что Git установлен, выполнив:
    git --version
  2. Вы увидите сообщение с установленной версией, например:
    git version 2.x.x

НАСТРОЙКА GIT

После установки Git выполните следующие шаги, чтобы настроить его для работы:

  1. Укажите email

    Email должен быть в формате i24s0000@students.it-college.ru, заменяя i24s0000 на ваш корпоративный логин:
    git config --global user.email "i24s0000@students.it-college.ru"

  2. Укажите имя пользователя

    Имя пользователя будет отображаться в ваших коммитах. Вместо "i24s0000" используйте свой корпоративный логин:
    git config --global user.name "i24s0000"

  3. Проверьте настройки

    Проверьте, что настройки применены корректно:
    git config --list

Вы должны увидеть строки с вашим корпоративным логином и email. Теперь Git готов к работе в колледже!

ОТПРАВКА ФАЙЛОВ В РЕПОЗИТОРИЙ GIT

  1. Подключитесь к серверу колледжа через OpenVPN. Это обязательный шаг перед любыми действиями с репозиторием.

  2. Укажите путь до рабочего стола:

    cd Desktop

  3. Клонируйте репозиторий (замените URL на ссылку вашего репозитория):

    git clone URL

  4. Перейдите в папку репозитория:

    cd Work
  5. Закиньте все нужны файлы в папку Work (также для удобства можете создать внутри дополнительные папки)

  6. Добавьте все файлы в индекс:

    git add .
    Команда git add . добавляет все измененные, новые и удаленные файлы из текущей директории и её вложенных папок в индекс Git. Это подготовительный этап перед созданием коммита.

  7. Создайте коммит (название коммита может быть любым, например, 111 или update):

    git commit -m "111"
    Коммиты — это «снимки» вашего проекта в определённый момент времени. Они позволяют фиксировать изменения, описывать их и при необходимости возвращаться к предыдущим версиям. Это как история вашего проекта, доступная для анализа и отката.

  8. Отправьте изменения в репозиторий:

    git push

Теперь ваши файлы успешно загружены в репозиторий!
