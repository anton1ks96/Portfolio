ЧТО ТАКОЕ VS CODE:

Visual Studio Code (VS Code) — это бесплатный редактор исходного кода, разработанный Microsoft. Он поддерживает множество языков программирования, имеет встроенную поддержку Git, мощную систему расширений и высокую настраиваемость. Благодаря своей гибкости и широким возможностям, VS Code стал одним из самых популярных инструментов среди разработчиков по всему миру.

УСТАНОВКА VISUAL STUDIO CODE

Установка VS CODE

Скачивание установочного файла

  1. Откройте браузер и перейдите на официальный сайт Visual Studio Code: https://code.visualstudio.com.
  2. На главной странице автоматически будет предложена версия для вашей операционной системы. Нажмите кнопку "Download for Windows" или выберите нужную версию для вашей системы (например, macOS или Linux).
  3. Дождитесь завершения загрузки установочного файла.

Установка VS Code на Windows

  1. Запустите скачанный установочный файл (обычно он называется VSCodeSetup-x64-x.xx.x.exe).
  2. Пройдите мастер установки, выполняя следующие шаги:
    • Лицензионное соглашение: Прочитайте и примите лицензионное соглашение, затем нажмите "Next".
    • Выбор папки установки: Оставьте путь по умолчанию или выберите другую папку, затем нажмите "Next".
    • Выбор дополнительных задач: Рекомендуется отметить опции "Add to PATH" и "Register Code as an editor for supported file types" для удобства использования, затем нажмите "Next".
    • Установка: Нажмите "Install" для начала процесса установки.
    • Завершение установки: После завершения установки убедитесь, что отмечена галочка "Launch Visual Studio Code", и нажмите "Finish" для запуска редактора.

Установка VS Code на MacOS

  1. Откройте скачанный файл (обычно это VSCode-darwin-stable.zip).
  2. Распакуйте архив и перетащите приложение Visual Studio Code в папку Applications.
  3. Запустите VS Code из папки Applications.
  4. (Опционально) Для удобного запуска из терминала можно установить команду code:
    • Откройте VS Code.
    • Нажмите Cmd+Shift+P для открытия палитры команд.
    • Введите shell command и выберите Shell Command: Install 'code' command in PATH.

Первичная настройка

Настройка пользовательских параметров

После установки и первого запуска VS Code рекомендуется выполнить первичную настройку для комфортной работы:

  1. Откройте настройки: Нажмите Ctrl+, (Windows) или Cmd+, (Mac) для открытия окна настроек. Либо используйте палитру команд (Ctrl+Shift+P или Cmd+Shift+P) и введите Preferences: Open Settings.
  2. Настройте основные параметры:
    • Theme: Выберите предпочитаемую тему оформления.
    • Font Size: Установите удобный размер шрифта.
    • Tab Size: Настройте размер отступов для кода.
    • Синхронизация настроек: Войдите в свою учетную запись Microsoft или GitHub для синхронизации настроек между устройствами.

Настройка темы оформления

  1. Откройте палитру команд (Ctrl+Shift+P или Cmd+Shift+P).
  2. Введите Color Theme и выберите Preferences: Color Theme.
  3. Выберите тему из предложенного списка или установите дополнительные темы через расширения.

Плагины

Установка расширений

Расширения (плагины) значительно расширяют функциональность VS Code. Для их установки выполните следующие шаги:

  1. Откройте панель расширений: Нажмите иконку Extensions на боковой панели или используйте сочетание Ctrl+Shift+X (Windows) / Cmd+Shift+X (Mac).
  2. Поиск расширений: В поле поиска введите название нужного расширения (например, Prettier, Material Theme).
  3. Установка: Нажмите кнопку Install рядом с выбранным расширением.
